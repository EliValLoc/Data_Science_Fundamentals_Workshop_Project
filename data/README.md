# Data

This project uses publicly available open data from the City of Zurich / OpenDataZurich.

The raw datasets are not included in this repository. They can be downloaded from the original OpenDataZurich data catalogue. Please refer to the respective dataset pages for the applicable licence terms and metadata.

Main data source:
- OpenDataZurich / City of Zurich Open Government Data Catalogue

Accessed for the original course project in Autumn Semester 2023.
